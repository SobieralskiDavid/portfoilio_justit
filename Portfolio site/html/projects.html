six step hover and typeWriter tutorial using HTML, CSS And JavaScript.

Step one

Here I have created script tags at the bottom of the body and linked it to my JS file, next I go into my script file and set up some variables to use in a function.

Step two

With the creation of script tags and the variables I set up the function typeWriter(), This function will set the innerHTML of the paragraph tag with the ID of #type3 and also set the speed of the outputted words.

Step three

Here we have created an onMouseOver event listener and attached it to an image, when we hover over this image the event listener will trigger some action that we will create in the following steps.

Step four

We can now add some style and transition to the hover event, I gave the image a ID of #mouse_over, then using CSS I created a rotation transformation effect with the @KeyFrames method, also including a border radius and Yellow border.

Step five

Finally we can style the empty paragraph that will be used to hold the output of the typeWriter function, I give the paragraph tag an ID of #type3, then I can style it with all the usual CSS styling properties.
